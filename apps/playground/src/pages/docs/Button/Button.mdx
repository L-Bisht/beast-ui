# Button

Buttons allow users to take actions, and make choices, with a single tap.

## Usage

Buttons are primarily used for actions and are the most common component in the system. They feature a distinct "Magnetic Press" interaction.

## Guidelines

- Avoid using multiple `filled` variant buttons on the same view.
- Use `ghost` buttons for secondary or less important actions.
