## What to build

Write Storybook stories for all display components: Text, Avatar, Badge, Tag, Divider, Tooltip. Each story demonstrates all prop variants.

## Acceptance criteria

- [x] Text story shows all sizes, weights, colors, truncation, and semantic elements
- [x] Avatar story shows image, initials fallback, default fallback, and all sizes
- [x] Badge story shows count, dot, max, position, and invisible states
- [x] Tag story shows all variant × color combinations, deletable, clickable, with icon
- [x] Divider story shows horizontal/vertical, solid/dashed
- [x] Tooltip story shows all positions and arrow/no-arrow
- [x] Each story uses Storybook controls for interactive exploration
- [x] Stories organized under "Display" category
