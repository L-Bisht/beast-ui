## What to build

Write Storybook stories for all layout components: Frame, Flex, Lattice, Container. Each story file should demonstrate all prop variants and common composition patterns.

## Acceptance criteria

- [x] Frame story shows padding, margin, and polymorphic `as` usage
- [x] Flex story shows direction, gap, align, justify, and wrap variants
- [x] Lattice story shows column counts, template strings, areas, and responsive layouts
- [x] Container story shows all maxWidth presets
- [x] Each story uses Storybook controls (args) for interactive prop exploration
- [x] Stories are organized under a "Layout" category in the sidebar
