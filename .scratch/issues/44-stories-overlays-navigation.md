## What to build

Write Storybook stories for overlay and navigation components: Dialog, Drawer, Menu, Toast, Tabs, Breadcrumb, Link.

## Acceptance criteria

- [x] Dialog story shows sizes, close behaviors, with form content
- [x] Drawer story shows all anchor positions, sizes
- [x] Menu story shows items with icons, shortcuts, groups, separators, disabled items
- [x] Toast story shows all severities, with action buttons, stacking behavior
- [x] Tabs story shows all variants, orientations, disabled tabs
- [x] Breadcrumb story shows separator customization, maxItems collapse
- [x] Link story shows external, underline modes, disabled, polymorphic usage
- [x] Stories organized under "Overlays" and "Navigation" categories
