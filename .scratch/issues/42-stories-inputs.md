## What to build

Write Storybook stories for all input components: Button, Input, Checkbox, Radio, Switch, Select (both APIs). Each story demonstrates all prop variants and interaction states.

## Acceptance criteria

- [x] Button story shows all variant × color × size combinations, icon modes, loading, disabled
- [x] Input story shows variants, sizes, adornments, error states, helper text
- [x] Checkbox story shows checked, unchecked, indeterminate, disabled, error
- [x] Radio story shows RadioGroup with orientation, disabled options, error
- [x] Switch story shows all sizes, colors, disabled
- [x] Select story shows props API usage and compound API usage side-by-side
- [x] Each story uses Storybook controls
- [x] Stories organized under "Inputs" category
